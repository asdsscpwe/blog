1. 
만약 코드가 ...
```bash
    int[] num = new int [6];
    int[] inputNum = {10,20,30,40,50,60,1};
```

이렇다면 첫번째 줄의 코드와 두번째 둘의 차이점은 뭘까?
두 코드의 차이점은 이러하다:
-> 첫번째 줄은 쓴 숫자 만큼의 저장 공간을 생성한 코드이고 두번째 줄에 있는 코드는 배열의 값을 저장한 코드이다...
